# Kotlin Spring Boot Learning App

A simple Kotlin Spring Boot application for learning purposes, demonstrating integration with Apache Kafka.

---

## Requirements

- [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/)
- Java 21 (used via Gradle toolchain)
- Internet connection to download Docker images
